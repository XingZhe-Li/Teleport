# Teleport
### A shortcut to jump between dirs
```cmd
Tp is a simple Directory Jumper (Python Edition)
        Syntax: +(name) (dir)   ==> Create a destination
        Syntax: -(name)         ==> Delete a destination
        Syntax: *(name)         ==> Check a destination
        Syntax: (name)          ==> Jump to destination
        Syntax: [list/*]        ==> Show all destinations
        Syntax: back            ==> Jump to where before jump
```
